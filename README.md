# SymptoCare 🏥

An AI-based medical diagnosis system that predicts diseases based on symptoms.

## Features
- Patient login system
- Symptom-based disease prediction
- Health monitoring (Temp, SpO2, Pulse)
- IoT integration (ESP32 + sensors)

## Tech Stack
- Python / ML
- Arduino / ESP32
- Blynk App

## Future Scope
- Improve accuracy using real datasets
- Mobile app integration
